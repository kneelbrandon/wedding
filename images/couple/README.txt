# images/couple/

Place your couple photo here.

Recommended filename: photo.jpg
Recommended size: 600×600 px or larger (square works best for the circular frame)
Supported formats: JPG, PNG, WEBP

To use it in the app, update index.html:
  1. Find the <img id="couple-photo"> tag
  2. Set src="images/couple/photo.jpg"
  3. Remove the photo-placeholder <div> below it
  4. Add style="display:block;" to the <img> tag

Or just click the circle in the browser to upload on the fly.
